# Spring-Boot-Crud
Um sistema web completo para gerenciamento de produtos, desenvolvido com Spring Boot no backend e HTML, CSS e JavaScript no frontend.

# Funcionalidades
CRUD Completo: Criar, Ler, Atualizar e Deletar produtos.
Interface Responsiva: Design moderno em modo escuro com validação em tempo real.
Busca Inteligente: Filtra produtos por nome diretamente na interface.
API RESTful: Backend estruturado com controllers, services e repositories.
Validação Robusta: Tanto no frontend quanto no backend (com anotações do Spring).

#Tecnologias
Backend: Java, Spring Boot, Spring Data JPA, H2.
Frontend: HTML5, CSS3, JavaScript.
Banco de Dados: H2.
Ferramentas: Maven, Git.

## Como Executar
1. Clone o repositório:
   ```bash
   git clone https://github.com/Mou-Spec/Spring-Boot-Crud.git
   ```

2. Execute o backend:
   - Abra o projeto em sua IDE (IntelliJ, Eclipse, VS Code).
   - Execute a classe `CrudSpringApplication.java`.
   - O servidor iniciará em `http://localhost:8080`.

3. Acesse o frontend:
   - Abra o arquivo `index.html` no navegador.
   - A interface se conectará automaticamente à API.
